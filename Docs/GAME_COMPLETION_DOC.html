1. Tổng Quan Sản Phẩm

The Senior Defender là game 2D top-down shooter/defender. Người chơi bảo vệ một server đặt ở giữa màn hình. Các enemy liên tục lao tới để phá server. Player phải di chuyển quanh khu vực phòng thủ, bắn hạ enemy, thu thập DataPack để tích điểm nâng cấp và sống sót qua nhiều wave cho đến khi thanh Build Progress đạt 100%.

Điều kiện thua rất rõ: server hết HP là thua ngay. Điều kiện thắng cũng rất rõ: giữ server sống cho tới khi Build Progress đạt 100%.
Core loop

Bảo vệ server trung tâm

Enemy spawn theo wave và lao vào server. Người chơi phải chặn đường, bắn hạ và ưu tiên enemy nguy hiểm.

Progression

DataPack và nâng cấp sinh tồn

Enemy chết thưởng DataPack. Người chơi dùng điểm này để nâng sát thương và tốc độ bắn, giúp sống sót qua wave sau.

Polish

Áp lực build 100%

Build Progress là đồng hồ chiến thắng. Càng gần 100%, enemy càng đông và feedback phải làm người chơi cảm thấy căng hơn.

2. User Flow Chi Tiết

  1. Main Menu: người chơi thấy tên game, nút Play, Guide, Quit. Guide phải nói rõ mục tiêu là bảo vệ server ở giữa.
  2. Vào GameScene: server nằm ở trung tâm. HUD hiện Server HP, Player HP, Build Progress, Wave và DataPack.
  3. Wave bắt đầu: enemy spawn từ ngoài vùng chơi và lao thẳng tới server để phá nó. Wave 1 nhẹ để người mới học cách bắn.
  4. Combat: người chơi dùng WASD/Arrow để di chuyển quanh server, chuột trái để bắn enemy, chuột phải để dùng ultimate.
  5. Nhận DataPack: enemy bị tiêu diệt cộng DataPack ngay. DataPack là điểm tài nguyên để nâng cấp sức mạnh.
  6. Nâng cấp: sau wave, game pause và mở Upgrade Panel. Người chơi ưu tiên nâng sát thương, tốc độ bắn, hoặc Firewall nếu server yếu.
  7. Wave sau khó hơn: enemy spawn dày hơn, có enemy nhanh, enemy né đạn và enemy trâu hơn để ép người chơi tận dụng upgrade.
  8. Kết thúc: nếu Server HP về 0 thì thua ngay; nếu Build Progress đạt 100% khi server còn sống thì thắng.

Flow 30 giây đầu

  • 0-5 giây: server ở giữa phải nổi bật; HUD Server HP và Build Progress phải dễ nhìn.
  • 5-15 giây: enemy đầu tiên lao về server từ hướng rõ ràng; người chơi bắn hạ và thấy DataPack tăng.
  • 15-30 giây: server có thể bị đánh nhẹ để người chơi hiểu mất server là thua ngay.

4. Chỉ Số Kỹ Thuật Cần Đo

Performance mục tiêu

  • 60 FPS trong gameplay thường.
  • >= 45 FPS khi có 100 enemy active.
  • Không thấy đứng khung hình khi spawn group.
  • GC Alloc trong combat càng thấp càng tốt.

Stability checklist

  • Không có NullReferenceException khi vào GameScene.
  • Restart sau Win/Lose không bị kẹt Time.timeScale = 0.
  • Enemy pooled không giữ animation chết.
  • Bullet pooled không giữ velocity cũ.
Kiểm tra bằng Unity Profiler: tạo 80-120 enemy active, xem CPU Usage, GC Alloc và spike khi enemy spawn/chết.

7. UI/UX Cần Hoàn Thiện

HUD trong game

  • Góc trái trên: Player HP.
  • Giữa trên: Build Progress + Wave.
  • Góc phải trên: DataPack.
  • Gần server hoặc dưới màn hình: Server HP.
  • Góc trái dưới: Ultimate cooldown.

Upgrade Panel

  • Hiện level hiện tại.
  • Hiện chỉ số trước và sau khi mua.
  • Hiện giá DataPack.
  • Upgrade chính nên xoay quanh sát thương, tốc độ bắn và khả năng giữ server sống.
  • Nếu không đủ điểm, hiển thị thiếu bao nhiêu DataPack.

Win/Lose UI

Màn hình kết quả nên hiện Bugs defeated, DataPack earned, Upgrades purchased, Player deaths, Server HP remaining hoặc Build Progress khi thua.

8. Effect Và Game Feel

Bắn súng

  • Muzzle flash tại firePoint.
  • Bullet trail nhẹ.
  • Random pitch âm thanh 0.95-1.05.
  • Camera shake rất nhẹ khi bắn burst.

Enemy hit/death

  • Flash sprite khi trúng đạn.
  • Hit particle nhỏ.
  • Death particle riêng cho từng enemy.
  • Popup DataPack: +5, +10, +25.

Server & Ultimate

  • Server flash đỏ khi bị đánh.
  • Low HP warning dưới 30%.
  • Ultimate shockwave có glow/sound/camera pulse.
  • Icon ultimate flash khi sẵn sàng.

11. Definition Of Done

  • Main Menu -> Game -> Upgrade -> Win/Lose -> Restart chạy trọn vẹn.
  • Không có runtime error trong Console.
  • Game ổn định với ít nhất 80 enemy active.
  • Mỗi enemy có vai trò riêng và dễ nhận biết.
  • Mỗi upgrade có tác dụng cảm nhận được.
  • UI đủ rõ để người mới tự chơi.
  • Effect và âm thanh phản hồi đủ mọi hành động quan trọng.
  • README có Unity version, controls, objective và cách chạy.
